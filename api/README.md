# Backend

Backend is written in Rust using Rocker framework and postgreSQL database. It's hosted on Shuttle-rs.

# Running

```
# Install cargo shuttle
cargo install cargo-shuttle

# Run locally
cargo shuttle run
```

# Deploying

```
Cargo shuttle deploy
```
